1.1二阶三阶行列式

1、二阶行列式

|a11 a12|
|a21 a22|

= a11a22 — a12a21

aij 中 i是行标，j是列标

从左上到右下的线叫主对角线 \

从右上到左下的线叫次对角线 /

2、三阶行列式

|a11 | a12| a13|

|a21 |a22 |a23|

|a31 |a32 |a33|

3,排列

定义：由1，2，3，…..n 组成一个有序数组叫n级排列，中间不能缺数

4、逆序：

定义：大数排在小数的前面

5、逆序数：

定义： 逆序的总数 4213 的逆序数为4 即 N（4213）=4

逆序数为奇数就是奇排列，为偶数就是偶排列

6、对换：

定义：交换两个数

一次对换奇偶性发生改变

1.2n阶行列式：

1、第一定义（按行展开）

|a11 | a12| a13|

|a21 |a22 |a23|

|a31 |a32 |a33|

= a11 a22 a33 + a12 a23 a31 + a13 a21 a32
— a13 a22 a31 —a12 a21 a33 —a11 a23 a32

列标：123 231 312

​ 321 213 132

行标取标准排列

列标取排列的所有可能，从不同行不同列取出三个元素相乘

符号由列标排列的奇偶性决定（偶为正，奇为负）

上三角行列式，下三角行列式，对角线行列式

2、第二定义（按列展开）

列标取自然排列，行标取所有可能，符号由行标排列的奇偶性决定（偶为正，奇为负）

3、第三定义（既不按行也不按列）

行标排列的逆序数加上列标排列的逆序数相加，偶为正，奇为负

2.2 矩阵运算

1、加减乘除

1、加减即两个矩阵对应元素相减，前提是两个矩阵式同型矩阵

2、数乘运算：乘整体的时候要与每个元素相乘

同理提公因子时每个元素都需要有——行列式则是一行提一次

3、矩阵相乘（重点）

前提：第一个矩阵的列数 = 第二个矩阵的行数

结果形状：结果行数= 第一个矩阵的行数
结果矩阵的列数 = 第二个矩阵的列数

性质：

1、矩阵相乘不满足交换律

2、矩阵相乘满足结合律

3、矩阵相乘满足分配律

4、k(AB) = (kA)B = A(kB)

注，分配律：
C(A + B) 和（A + B）C 不一样

口诀： 中间相等，取两头

e.g.:A3*4 B4*5 = C3*5

推理关系：

  1. AB = 0 != > A = 0 或 B = 0

  2. AB = AC ,A != 0 !=> B = C

  3. AB != BA ,AB 有意义，BA不一定有意义

若AB是可交换的相乘矩阵，那么AB一定都是方阵

与单位阵（E）相乘 矩阵不变 即AE= A

三、对称矩阵和反对称矩阵

对称矩阵

1、以主对角线为轴，上下对应相等为对称矩阵 aij = a ji (主对角线无要求)

重点： AT = A

性质：

若A，B同阶对称，则他们之间的和、差、和数乘依然是对称的，但是其乘积一般是不对称的

定理一：

A，B对称，且AB对称 《==》 AB可交换

反对称矩阵：

2、以主对角线为轴且主对角线全为0，上下对应为相反数为反对称矩阵 aij = -aji

重点：AT = -A

性质：

若A，B同阶反对称，则他们之间的和、差、和数乘依然是对称的，但是其乘积一般是不对称的

谨记：永远不要把矩阵放在分母上

1、方阵的行列式

矩阵有多个属性，行列式属于矩阵的一个属性

性质一：

| AT| = |A| 来自：行列式转置值不变 的性质

性质二：

常见出错点
|kA| = kn |A| (n为方阵的阶数)

2、伴随矩阵

前提：只有方阵才有伴随矩阵

3、逆矩阵的概念

令A为n阶方阵，若存在n阶方阵B，使得

AB = BA = I

则称A为可逆矩阵，B为A的逆矩阵

记为 A-1 = B

定理一：

设A是可逆矩阵，那么A的逆矩阵是唯一的

定理二：

设A,B均为n阶可逆矩阵，数b 不等于0

1、A-1 可逆，且A-1-1 = A //可逆矩阵的逆矩阵可逆

2、 bA 可逆，且 （bA）-1 = 1/b A-1

3、AB可逆，且（AB）-1 = B-1A-1

4、AT 可逆，且（AT）-1 = (A-1)T

若AB = 0 ,则A,B不同时可逆，有一个为零矩阵

2.5高斯消元和矩阵的初等变换

一、行阶梯矩阵：

特点：
1、每一行的非零首元出现在上一行非零首元的右边

2、所有的零行位于矩阵的下端

特别的：

若每一个非零行的非零首元都是1，且其所在列的其余元都是零，称为简化的行阶梯矩阵

注：行阶梯矩阵Am*n非零行的行数为r
(r<=m)且（r<=n）

二、矩阵的行初等变换：

矩阵的行（列）初等变换：

1、交换矩阵的两行

2、用一个非零常数乘矩阵的某行

3、某一行乘以常熟后加到零一行上去

三、高斯消元法：

对AX = b的增广矩阵实施行初等变换，化其为行阶梯型来求解方程

d是增广矩阵的元

r是非零行的行数（也是原方程组有效方程的个数）

n为方程的未知量的个数

与这个矩阵对应的非齐次线性方程组与AX = b是同解方程组

关于解的情况(Am*nX = b解的存在性定理)：

如果dr+1 = 0,则AX = b有解,反之则无解

若有解：

则当 r = n 时，，有唯一解

​ 当 r < n时，有无穷多个解

定理的推论：

推论一:

AX = 0 必有解，

且当 r = n 时，有唯一解（仅有零解）

​ 当 r < n 时，有无穷多个解—>充要条件是 有非零解

推论二:

AX = 0, A 为 m*n矩阵，且有 m < n ,则AX = 0 必有非零解

矩阵的等价：

A 经过有限次初等变换之后变成了 B ，那么就说A与B等价

1）反身性：任何一个矩阵和自身等价

2）对称性： A和B等价，即为B和A等价

3）传递性：A和B等价，B和C等价，那么A和C等价
