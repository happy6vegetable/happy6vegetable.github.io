1.1二阶三阶行列式

1、二阶行列式

|a11 a12|
|a21 a22|

= a11a22 — a12a21

aij 中 i是行标，j是列标

从左上到右下的线叫主对角线 \

从右上到左下的线叫次对角线 /

2、三阶行列式

|a11 | a12| a13|

|a21 |a22 |a23|

|a31 |a32 |a33|

3,排列

定义：由1，2，3，…..n 组成一个有序数组叫n级排列，中间不能缺数

4、逆序：

定义：大数排在小数的前面

5、逆序数：

定义： 逆序的总数 4213 的逆序数为4 即 N（4213）=4

逆序数为奇数就是奇排列，为偶数就是偶排列

6、对换：

定义：交换两个数

一次对换奇偶性发生改变

1.2n阶行列式：

1、第一定义（按行展开）

|a11 | a12| a13|

|a21 |a22 |a23|

|a31 |a32 |a33|

= a11 a22 a33 + a12 a23 a31 + a13 a21 a32
— a13 a22 a31 —a12 a21 a33 —a11 a23 a32

列标：123 231 312

​ 321 213 132

行标取标准排列

列标取排列的所有可能，从不同行不同列取出三个元素相乘

符号由列标排列的奇偶性决定（偶为正，奇为负）

上三角行列式，下三角行列式，对角线行列式

2、第二定义（按列展开）

列标取自然排列，行标取所有可能，符号由行标排列的奇偶性决定（偶为正，奇为负）

3、第三定义（既不按行也不按列）

行标排列的逆序数加上列标排列的逆序数相加，偶为正，奇为负

2.1矩阵的概念

1、矩阵与行列式的区别

1、矩阵的本质是一个数表，而行列式的本质是一个数

2、矩阵的行列数不一定相等，而行列式必须相等

2、几个概念：

实矩阵（全为正数）；

复矩阵（全为复数）；

负矩阵 ；A -A (和原矩阵相反)

单位阵：En

零矩阵：元素全是0

行矩阵，列矩阵：只有一行或一列的矩阵

方阵：行列数相等

对角矩阵：除了某一条对角线外元素均为0的矩阵

2.2 矩阵运算

1、加减乘除

1、加减即两个矩阵对应元素相减，前提是两个矩阵式同型矩阵

2、数乘运算：乘整体的时候要与每个元素相乘

同理提公因子时每个元素都需要有——行列式则是一行提一次

3、矩阵相乘（重点）

前提：第一个矩阵的列数 = 第二个矩阵的行数

结果形状：结果行数= 第一个矩阵的行数
结果矩阵的列数 = 第二个矩阵的列数

性质：

1、矩阵相乘不满足交换律

2、矩阵相乘满足结合律

3、矩阵相乘满足分配律

4、k(AB) = (kA)B = A(kB)

注，分配律：
C(A + B) 和（A + B）C 不一样

口诀： 中间相等，取两头

e.g.:A3*4 B4*5 = C3*5

推理关系：

  1. AB = 0 != > A = 0 或 B = 0

  2. AB = AC ,A != 0 !=> B = C

  3. AB != BA ,AB 有意义，BA不一定有意义

若AB是可交换的相乘矩阵，那么AB一定都是方阵

与单位阵（E）相乘 矩阵不变 即AE= A

4、矩阵的幂运算

1）规定：A0 = E

2）（AB）k != AkBk

3)(A + B )2 != A2 +2AB + B2

(A-B)2 同理

三、对称矩阵和反对称矩阵

对称矩阵

1、以主对角线为轴，上下对应相等为对称矩阵 aij = a ji (主对角线无要求)

重点： AT = A

性质：

若A，B同阶对称，则他们之间的和、差、和数乘依然是对称的，但是其乘积一般是不对称的

定理一：

A，B对称，且AB对称 《==》 AB可交换

反对称矩阵：

2、以主对角线为轴且主对角线全为0，上下对应为相反数为反对称矩阵 aij = -aji

重点：AT = -A

性质：

若A，B同阶反对称，则他们之间的和、差、和数乘依然是对称的，但是其乘积一般是不对称的
